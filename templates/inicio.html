API

API Vale do Paraíba


Buscar por cidade

Para consultar uma cidade específica, informe a cidade ao final: https://api-valeparaiba.herokuapp.com/valedoparaiba/coordenadas/nome da cidade

Exemplo usando o município Queluz, retorna um Json:
lat: -22.5375
lon: -44.7746

Segue um exemplo de código em Python para realizar essa requisição:

from requests import get
from json import loads

resposta = get("https://api-valeparaiba.herokuapp.com/valedoparaiba/coordenadas/Queluz/")
conteudo = loads(resposta.content)

print(conteudo)

Buscar todas as cidades

Para consultar todas as cidades: https://api-valeparaiba.herokuapp.com/valedoparaiba

Ao consultar todas as cidades, retorna uma lista em que cada índice possui as chaves cidade, lat e lon, exemplo:
cidade: "CRUZEIRO"
lat: -22.5765
lon: -44.9634

Segue um exemplo de código em Python para realizar essa requisição:

from requests import get
from json import loads

resposta = get("https://api-valeparaiba.herokuapp.com/valedoparaiba/")
conteudo = loads(resposta.content)

print(conteudo)
